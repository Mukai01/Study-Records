# 1. 初期設定

```sh
#初期設定
git config --global user.name "Nakamukai1"
git config --global user.email "登録メールアドレス"

#vs codeを登録
git config --global core.editor "code --wait"

#設定を確認
git config user.name
git config user.email
git config core.editor

#以下でまとめて確認を行うことも可能
git config --list

#結果は以下に保存されている
#catはファイルの内容を確認
cat ~/.gitconfig
```
